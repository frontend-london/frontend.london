About me

Over the years I learned many programming languages - Basic, Pascal, Delphi, HTML, CSS, PHP, SQL, Javascript, Ada, C++, Assembler, Python, Java and much more.

For more than seven years I'm working as a Web Developer, using languages HTML, CSS, Javascript, PHP and SQL to my everyday work.

Throughout this period the trends in the front end changed several times. At the beginning I used for this purpose languages css 2, html 4, and the library prototype.js, then library jQuery, and currently I’m writing responsive websites using languages html 5, css 3, and frameworks such as Backbone.js, Bootstrap for css or Jasmine for testing.

For over five years I dealt with the backend in php and I wrote a few large sites using Zend, Symfony and Code Igniter framework such as webshop with music smashintracks.com.
